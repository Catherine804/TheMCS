# TheMCS: The Sheep App (SheepTrack)

> **Virtual Pet Productivity Tracker App**  
Helping people accomplish their goals with a virtual pet.
---

## Overview
**The Sheep App** is a productivity app that tracks whether the user is accomplishing their daily goal or task by giving them a virtual sheep to take care of. This sheep gets happy or sick depending on the user's accomplishment of the task.
